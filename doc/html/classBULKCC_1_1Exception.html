Detailed Description

Exception class for BULKCC library errors.

The Exception class provides a simple mechanism for reporting errors within the BULKCC library. It stores an error message that can be retrieved using the what() method.

Basic Usage

try {
// Code that might throw an exception
throw BULKCC::Exception("Something went wrong");
} catch (BULKCC::Exception& e) {
std::cout << "Error: " << e.what() << std::endl;
}
