Detailed Description

template<class T>
class BULKCC::ObjTemplate< T >

Template class for type-specific stack nodes.

The ObjTemplate class is a concrete implementation of the Obj base class that can store objects of a specific type T. It provides type-safe storage and retrieval of objects within the stack structure.

Template Parameters
TThe type of object to store in this node

Overview

Each ObjTemplate node can hold one object of type T. The object is stored as a member variable and can be added, fetched, or erased through the provided methods.

Basic Usage

This class is typically not used directly. Use the Controller class to create and manage stacks:

BULKCC::Controller controller;
MyClass* obj = new MyClass();
BULKCC::Obj* stack = controller.add<MyClass*>(obj);
