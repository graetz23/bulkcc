Detailed Description

Abstract base class for stack nodes.

The Obj class is the abstract base class for all stack nodes in the BULKCC library. It provides the fundamental linked-list structure for storing objects in a stack.

Design

Each stack node contains:

  • A pointer to the next node in the stack (_nextObj)
  • Virtual methods for type identification and object management

This class cannot be instantiated directly. Use ObjTemplate<T> to create concrete stack nodes that can store objects of a specific type.

Basic Usage

This class is typically not used directly. Instead, use the Controller class to manage stacks:

BULKCC::Controller controller;
BULKCC::Obj* stack = controller.add<MyType*>(myObject);
